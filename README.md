# Tax Loss Harvesting Tool

This is a React-based UI built for the KoinX frontend assignment. It simulates how tax loss harvesting works by letting users select assets and see how their capital gains change.

## Features

* Shows pre and post harvesting capital gains
* Select/deselect assets to update calculations
* “Select All” functionality
* Displays tax savings dynamically
* Clean and responsive UI

## Tech Stack

* React (Vite)
* Tailwind CSS
* Mock APIs

## How to Run

```bash
npm install
npm run dev
```
